# PlatePulse

A fully responsive, mobile-first diet planning application designed to revolutionize how users manage their nutrition across all diet types.

## 🌟 Overview

PlatePulse is an enterprise-grade diet planning platform that combines flexibility, intelligence, and beautiful design to help users achieve their nutritional goals. Whether you're following vegan, keto, vegetarian, or custom diet combinations like "vegan keto," PlatePulse adapts to your unique needs.

## ✨ Key Features

### Intelligent Meal Planning
- **Infinite Meal Generation**: AI-powered meal planning that creates exciting, varied recipes
- **Flexible Configuration**: Customize meals per day, calorie targets, macronutrient distribution
- **Profile Combinations**: Combine multiple diet profiles (e.g., vegan + keto) for personalized plans
- **Recipe Database**: Extensive collection of recipes with detailed nutritional information

### Comprehensive Tracking
- **Calorie & Macro Tracking**: Real-time monitoring with visual progress indicators
- **Motivational System**: Gamified achievements and progress celebrations
- **Barcode Scanner**: Instantly scan products to verify compatibility with your diet profile
- **Meal History**: Track your nutritional journey over time

### Export & Integration
- **Multiple Export Formats**: PDF, JSON, CSV, API access
- **Shareable Plans**: Export meal plans for team or family coordination
- **API Integration**: Developer-friendly API for custom integrations

### AI-Powered (Optional)
- **LLM Integration**: Connect custom language models for personalized meal suggestions
- **MCP Support**: Model Context Protocol integration for enhanced AI capabilities
- **Smart Recommendations**: Intelligent meal suggestions based on preferences and history

## 🎨 Design Philosophy

- **Mobile-First**: Optimized for mobile devices with responsive desktop support
- **Vibrant Color Palette**: Energetic, motivating colors that make nutrition tracking enjoyable
- **Accessibility First**: WCAG AA compliant with keyboard navigation and screen reader support
- **Cross-Platform**: Available on Desktop (Tauri), Web (Next.js), and Mobile

## 🚀 Tech Stack

### Frontend
- **Framework**: Next.js 14+ (React Server Components)
- **Styling**: Tailwind CSS
- **Desktop Shell**: Tauri 2.0
- **State Management**: Zustand + React Query

### Backend
- **Framework**: NestJS 10+
- **API**: REST + GraphQL
- **Database**: PostgreSQL
- **Cache**: Redis

### AI/ML
- **LLM Integration**: OpenAI, Anthropic, Custom Models
- **MCP Support**: Model Context Protocol
- **Recommendation Engine**: Custom algorithm + LLM hybrid

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/MountainLabsDE/DietPlanner.git
cd DietPlanner

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
